# Branding Policy

The OpenDeck source code is licensed under `GPL-3.0-or-later`.

That license covers the code, but it does not grant trademark or brand rights.

This project reserves the following identifiers for source-identification and anti-confusion purposes:

- `OpenDeck`
- `Linux Stream Deck App`
- the OpenDeck app icon, logos, and other project branding assets

You may:

- refer to the project by name when discussing, packaging, reviewing, or linking to the official project
- redistribute exact, unmodified copies of the official project with the original name and branding
- say that your plugin, theme, or fork is compatible with OpenDeck

You may not:

- present a modified fork as the official OpenDeck project
- use the OpenDeck name, icon, or branding in a way that implies endorsement, authorship, or official status for a modified build
- remove upstream copyright and license notices

If you distribute a modified fork, you should:

- clearly mark it as modified and unofficial
- keep the original copyright and license notices intact
- rename the app or clearly distinguish your branding if there is any chance of confusion

If you want to use the official branding for something outside normal reference to the project, ask the maintainers first.
